💾 Backup & Security

Protect your data with regular backups. Export manual backups or restore from a backup file.

⚠️ Important: Regular Backups

Export a backup every Friday! This protects against accidental data loss, system errors, or unauthorized changes. Store backups in Google Drive, email them to yourself, or save on multiple devices.

📤 Export Backup

Download a complete backup of all system data (teachers, students, tickets, settings, history). This creates a JSON file you can restore from later.

Last automatic backup: Never

📥 Restore from Backup

Upload a previously exported backup file to restore all data. Warning: This will replace ALL current data!

🤖 Automatic Backups

The system automatically creates daily backups to Firebase cloud storage. These backups:

  • Run once per day when anyone logs in
  • Store in Firebase (separate from main data)
  • Keep last 30 days of backups
  • Protect against accidental deletions
